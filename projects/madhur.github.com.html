http://madhur.github.com

This current site built with Jekyll is a project in itself. It has come a long way since I started in this 2011.

Some of the unique features of this site are:

  • Static site integration with Jekyll
  • Integration with Disqus
  • CSS design using Blueprint CSS framework
  • Integration with Gibhub API to pull commits and issues using Octokit
  • Syntax highlighting with Pygments
  • Integration with Fancybox
  • Scalebable CSS icons from FontAwesome
  • Site and source code hosting on github
  • Custom Tag Cloud Plugin

This site is heavily inspired and borrows some of the plugins and design from Carl Boettiger and Damien du Toit and Bilal Syed Hussain
