How this site is developed

This website is created with Jekyll, a static website generator written in Ruby. The pages are written in Markdown using plain notepad. You can have a look at the source code on GitHub. The template system it uses Liquid. (The syntax-highlighting, is generated by Pygments. I push all of these out to my GitHub repository using the version control system git.
